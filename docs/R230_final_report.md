## 六、关键参数模式

### 共同最优特征

| 参数 | M30 最常见值 | H4 最常见值 |
|------|-------------|-------------|
| Trail Activate | **0.15 ATR** (11/12) | **0.15 ATR** (12/14) |
| Trail Distance | **0.04 ATR** (11/12) | **0.04 ATR** (12/14) |
| Max Hold | **24 bars** (8/12) | **15-20 bars** (10/14) |
| SL Range | 2.0-3.0 ATR | 1.5-4.0 ATR |
| TP Range | 3.0-12.0 ATR | 2.0-6.0 ATR |

### 关键发现

1. **激进尾随止盈 (0.15/0.04)** 是几乎所有策略的最优选择，说明在黄金市场中快速锁定利润比追逐趋势更有效
2. **短持仓周期** (M30: 24 bars=12h, H4: 15-20 bars=2.5-3.3天) 明显优于长持仓
3. **SL:TP 比例趋近 1:1 或 1:2**，而非传统的 1:3+
4. **所有策略在所有时代均盈利**，说明这些策略在不同市场环境下具有强鲁棒性

## 七、风险提示

1. **Sensitivity Gate 未全部通过**: H4有7个策略在参数敏感性测试中FAIL，说明这些策略对参数选择较敏感，实盘时需注意参数稳定性
2. **Sharpe 值偏高**: 部分策略 Sharpe>8 可能暗示过拟合风险，建议:
   - 对比 baseline vs optimized Sharpe 的差距
   - h4_rsi baseline=37.5 → optimized=10.5，反而下降，说明 baseline 是过拟合
3. **交易成本假设**: 测试使用 spread=0.30，实盘可能更高，但 slippage test 在 spread=0.50 仍全部 PASS
4. **M30数据量大但单笔PnL小**: M30策略平均每笔利润较低，需要注意滑点对实际表现的影响
